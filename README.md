Top Courses – React Application

Top Courses is a modern and interactive React application that showcases high-quality courses across multiple categories such as Development, Business, Design, and more.  
The app features dynamic course rendering, reusable components, and category-based filtering to provide a smooth and intuitive browsing experience.



 Live Demo :-https://supriyode2002.github.io/top-courses/


 Features:-
 
1. Category-based course filtering  
2. Dynamic rendering of course cards  
3. Clean and modern UI  
4. Built with reusable React components  
5. Fully responsive layout  
6. Smooth browsing experience  
7. Data-driven structure for easy scalability  


Tech Stack:-

React.js – Component-based UI  
JavaScript – Logic & dynamic rendering  
CSS – Styling and layout  
Create React App – Build setup  
GitHub Pages – Deployment
